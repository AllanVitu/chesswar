Mobile

Transformer ChessWar en app mobile.

Ajoutez l'app à l'écran d'accueil pour jouer en plein écran.

iOS

iPhone (Safari)

  1. Ouvrez ChessWar dans Safari.
  2. Partage → Sur l'écran d'accueil.
  3. Validez pour créer l'icône.
Android

Android (Chrome / Edge)

  1. Ouvrez ChessWar dans le navigateur.
  2. Menu ⋮ → Installer l'application.
  3. Lancez en plein écran.
